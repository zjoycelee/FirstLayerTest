# Mini Basket Bag

## Description
This is a mini basket-style bag. After printing, the handles on both sides are inserted through slots to assemble the basket. It can be used as a decoration for dolls or for storing small items like candy.

- **Print Time:** 29 minutes
- **Material Usage:** 6g
- **Layer Height:** 0.1mm
- **Total Height:** 0.2mm
- **Finished Dimensions:**
  - **Base Length:** 85 mm
  - **Body Height:** 60 mm
- The model can be scaled as needed. Adjustments to hole compensations for the X and Y axes are required after scaling.

### For A1 Mini Printer

- **Finished Dimensions:**
  - **Base Length:** 60.5 mm
  - **Body Height:** 42 mm

### Slicing Settings
- **Bottom Pattern:** Set to **Concentric** to prevent corner tearing after assembly.
- **Top Pattern:** Set to **Lines** to achieve a smoother surface finish.
- **Layer Height:** 0.1mm for better strength.

## Files
- `miniBag.stl`: The STL file for printing the original model.
- `miniBag.3mf`: The 3MF file for easy printing with Orca or Bambu Studio.
- `miniBag(a1mini).3mf`: A pre-scaled 3MF file for printing with the A1 Mini printer.

## How to Assemble
1. After printing, insert the handles into the slots on both sides.
2. Adjust the handles as needed to create a miniature basket.

## Design Philosophy
The design is optimized for 3D printing, ensuring quick and efficient prints while providing a practical and decorative result.
